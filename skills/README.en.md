# Skills

[中文](README.md) | English

This directory is the human-facing skill entry point for the repository.

The current structure is intentionally split into two layers:

- `skills/*.md`: short Chinese skill cards for quick browsing.
- `skills/full/<skill>/`: public full skill packages, usually containing `SKILL.md` and sometimes extra `references/`, `scripts/`, or `README.md` files.

If you want the shortest route into the most representative items, start with:

- [`paper-review-pipeline`](paper-review-pipeline.md)

- [`collaborating-with-codex`](collaborating-with-codex.md)
- [`prompt-polisher`](prompt-polisher.md)
- [`writing-anti-ai`](writing-anti-ai.md)

## Research and Writing

- [`academic-paper-helper`](academic-paper-helper.md) / [`full`](full/academic-paper-helper/): LaTeX, BibTeX, structure templates, and writing scaffolding for AI/ML papers.
- [`paper-review-pipeline`](paper-review-pipeline.md) / [`full`](full/paper-review-pipeline/): a local paper review workflow with issue grading and revision guidance.
- [`paperreview`](paperreview.md) / [`full`](full/paperreview/): bridge to `paperreview.ai` for an external second opinion on near-final PDFs.
- [`question-refiner`](question-refiner.md) / [`full`](full/question-refiner/): refine a vague research topic into a structured deep-research brief.
- [`writing-anti-ai`](writing-anti-ai.md) / [`full`](full/writing-anti-ai/): remove AI writing patterns while keeping the author’s actual voice.
- [`timestamped-video-summary`](timestamped-video-summary.md) / [`full`](full/timestamped-video-summary/): transform timestamped transcripts into strict, structured summaries and optionally export them to PDF.
